# Sky Star Family Restaurant & Hotel

Welcome to the Sky Star Family Restaurant & Hotel official website repository. This application serves as a modern, responsive, and aesthetically pleasing digital storefront for our premium hospitality services in Karoli, Haryana. 

## 🌟 Overview

The Sky Star website is designed to provide guests with an effortless experience for:
- Viewing our diverse, tantalizing food menu.
- Exploring our luxurious bookable rooms and event halls.
- Getting in touch instantly via direct phone calls or WhatsApp integration.
- Navigating our story and gallery to see the ambiance before visiting.

## 🚀 Technologies Used

This project was built with a modern tech stack to ensure high performance and seamless user experience across all devices:

- **[React](https://reactjs.org/)** (v18+) - Component-based architecture
- **[TypeScript](https://www.typescriptlang.org/)** - For robust, type-safe code
- **[Vite](https://vitejs.dev/)** - Next-generation frontend tooling and bundling
- **[Tailwind CSS](https://tailwindcss.com/)** - Utility-first CSS framework for rapid UI development
- **[shadcn/ui](https://ui.shadcn.com/)** - Beautifully designed, accessible components
- **[Lucide React](https://lucide.dev/)** - Beautiful, consistent icons

## 🎯 Features

- **Dynamic Interactive Menu**: An integrated image slider for previewing the full menu pages natively.
- **Fully Responsive**: Flawless layout and typography adjustments for mobile, tablet, and desktop viewing. 
- **Modern Animations**: Smooth scroll behaviors, fade-ins, and elegant hover states.
- **Call-to-Action Integrations**: Instant connection buttons for placing orders via WhatsApp or calling the reception directly.

## 💻 Getting Started

### Prerequisites

Ensure you have [Node.js](https://nodejs.org/) (version 16 or newer) installed.

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/yourusername/sky-star-main.git
   cd sky-star-main
   ```

2. Install the dependencies:
   ```bash
   npm install
   ```

3. Start the development server:
   ```bash
   npm run dev
   ```

4. Open your browser and navigate to `http://localhost:5173` to see the application in action.

### Building for Production

To create a production-ready build:

```bash
npm run build
```
This command compiles the project into the `dist` folder, optimizing all assets for deployment. You can test the built application locally using:

```bash
npm run preview
```
